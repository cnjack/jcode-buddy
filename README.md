# 流光 — 智能桌面伴侣

流光是一款摆放在桌面上的智能信息终端。3.49 英寸长条形屏幕横向放置，一眼即可获取时间、天气和今日日程。无需主动操作，信息始终呈现在眼前。

---

## 产品形态

流光采用 3.49 英寸电容触摸屏，172 × 640 长条形横向布局，搭配深色主题 UI，适合放置在显示器下方或桌面任何角落。支持 USB 供电和锂电池供电，既可常驻桌面，也可随身携带。

---

## 界面布局

### 时钟页

屏幕从上到下分为四个区域，所有信息一屏展示，无需翻页或滑动：

```
┌────────────────────── 640px ──────────────────────┐
│                    时钟区域                         │
│    星期三  4月18日                                    │
│          14:35     ◜ 弧形秒针进度环                  │
│            47                                       │  130px
├───────────────────────────────────────────────────┤
│  局部多云    22°C              湿度 50%              │   40px
├───────────────────────────────────────────────────┤
│  日程                                               │
│  09:00  晨会                                        │
│  11:30  午餐                                        │   110px
│  14:00  代码评审                                     │
│  17:30  下班                                        │
├───────────────────────────────────────────────────┤
│              ▓▓▓▓▓▓░░░░░░  今日已过 60%            │   24px
└───────────────────────────────────────────────────┘
```

#### 时钟区域

屏幕最上方，占据最大面积。中央以 48 号字体显示当前时分（`14:35`），外圈包裹一圈弧形秒针进度环，每秒顺时针旋转一格，60 秒走满一圈。秒数数字以强调色显示在时间下方。顶部显示中文日期和星期（`星期三  4月18日`）。

#### 天气条

紧贴时钟下方的一条半透明磨砂面板。左侧显示天气描述（如「局部多云」），中间显示温度（`22°C`），右侧显示湿度（`湿度 50%`）。首次启动时显示「加载中...」，连接网络后自动填充真实数据。天气条还会随 BLE 猫咪状态联动变化。

#### 日程面板

天气条下方的磨砂面板，列出今日所有日程事件。每条日程左侧是时间（强调色），右侧是事件名称。面板底部有一条横向进度条，实时显示「今日已过 XX%」，直观感知一天过去了多少。

#### 今日进度条

屏幕最底部，一条渐变进度条配百分比文字，随时间推移自动增长。

### 蓝牙页

按 BOOT 键切换到蓝牙页，左侧显示像素猫咪宠物，右侧显示 BLE 消息列表：

```
┌──── 100px ────┬──────────── 540px ────────────────┐
│  [连接状态 ●]  │                                     │
│               │  消息 1（最新，最亮）                  │
│   🐱 像素猫    │  消息 2                              │
│   (动画)      │  消息 3                              │  172px
│   Z z z      │  消息 4                              │
│   [state]    │  消息 5（最旧，最淡）                  │
└───────────────┴─────────────────────────────────────┘
```

## 多页切换

流光支持两个页面，按 BOOT 键一键切换：

| 页面     | 内容                                   |
| -------- | -------------------------------------- |
| 时钟页   | 时钟 + 天气 + 日程 + 今日进度          |
| 蓝牙页   | 像素猫咪宠物 + BLE 消息列表            |

### 蓝牙页

蓝牙页通过 BLE NUS（Nordic UART Service）与手机或其他设备配对，设备广播名称为 `JCODE-XXXX`。配对后，设备可接收 JSON 指令控制猫咪状态和显示消息。

**猫咪动画状态：**

| 状态       | 触发命令       | 动画效果                     |
| ---------- | -------------- | ---------------------------- |
| 睡觉       | `sleep`        | 闭眼 + 飘浮 Zzz              |
| 空闲       | `idle`         | 睁眼 + 尾巴摆动 + 定期眨眼   |
| 工作中     | `working`      | 摇头 + 冒汗                  |
| 注意       | `attention`    | 瞪大眼 + 背景闪烁            |
| 完成       | `complete`     | 开心表情 + 弹跳 + 爱心飘出   |

**消息显示：** 手机发送 `{"cmd":"heart","val":"你好"}` 格式的 JSON，猫咪右侧会显示最近 5 条消息，按时间倒序渐隐。

右上角绿色圆点指示 BLE 连接状态。

## 交互方式

### 触摸

屏幕支持电容触摸，用于未来的交互扩展。

### 按键操作

| 操作         | 效果                                  |
| ------------ | ------------------------------------- |
| 按一下 BOOT  | 切换页面：时钟页 ↔ 蓝牙页             |

### 自动息屏

| 状态       | 触发条件       | 表现                     |
| ---------- | -------------- | ------------------------ |
| 正常显示   | —              | 全亮度运行               |
| 自动变暗   | 无操作 60 秒后 | 亮度降至最低（PWM 30）   |
| 自动关屏   | 无操作 3 分钟后 | 屏幕完全关闭             |

### 摇晃唤醒

息屏状态下，拿起设备轻轻摇晃即可唤醒屏幕。内部六轴传感器检测加速度变化，超过阈值（2.5g）时自动恢复显示。

## 启动流程

```
插电开机
  │
  ▼
显示启动画面 — 深色背景中央显示「流光」标题，下方出现旋转加载动画
  │
  ▼
自动连接 Wi-Fi
  │
  ▼
NTP 时间同步成功 → 将网络时间写入 RTC
  │
  ▼
销毁启动画面 → 创建主界面（时钟页 + 蓝牙页同时创建）
  │
  ▼
初始化 BLE NUS 服务 → 开始广播（设备名 JCODE-XXXX）
  │
  ▼
15 秒后开始拉取天气数据，之后每 30 分钟自动刷新
```

启动画面在整个网络初始化期间持续显示，用户无需等待或做任何操作，时间同步完成后自动无缝切换到主界面。蓝牙页默认隐藏，按 BOOT 键可随时切换查看。

## 数据更新机制

| 数据     | 更新频率   | 触发方式                       |
| -------- | ---------- | ------------------------------ |
| 时钟     | 每秒       | 从 RTC 读取，持续刷新          |
| 秒弧线   | 每秒       | 跟随秒数 0-59 循环            |
| 天气     | 每 30 分钟 | 后台 HTTP 请求，完成后刷新 UI  |
| 今日进度 | 每秒       | 根据当前时分计算百分比         |
| 日程     | 静态       | 代码中预置，开机时创建         |
| BLE 消息 | 实时       | BLE NUS 接收 JSON 指令         |
| 猫咪动画 | ~12 FPS    | LVGL 定时器驱动，状态由 BLE 控制 |

所有数据更新在后台线程完成，UI 刷新通过互斥锁保护，不会出现闪烁或撕裂。

## 配色方案

流光采用深色夜间主题，所有面板均为半透明磨砂质感：

| 角色       | 颜色       | 用途                         |
| ---------- | ---------- | ---------------------------- |
| 背景       | `#1A1A2E`  | 深蓝黑，整个屏幕底色         |
| 面板       | `#16213E`  | 稍浅的深蓝，卡片区域         |
| 强调色     | `#E94560`  | 珊瑚红，秒弧线、日程时间     |
| 主文字     | `#FFFFFF`  | 纯白，时钟数字、日程标题     |
| 次要文字   | `#8892B0`  | 灰蓝，日期、湿度             |
| 进度条底色 | `#0F3460`  | 深蓝，今日进度条背景         |

## 开发

```bash
idf.py build && idf.py -p COM7 flash   # 编译并烧录
idf.py -p COM7 monitor                 # 串口监视器
```

配置项见 `main/user_config.h`（Wi-Fi、天气城市、息屏超时、唤醒灵敏度等）。
